# 通信手法をPlugableにする

デバイスごとに利用可能な通信手法は異なり、また、新しい仕様が短い期間で生まれてくる。
このような状況に対応するためには、通信手法の部分を可能な限り独立したモジュールとしておくことが必要となる。
